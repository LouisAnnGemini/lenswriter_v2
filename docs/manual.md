# LensWriter 功能说明书

LensWriter 是一款专为作家和创作者设计的非线性写作工具，它通过独特的“透镜 (Lens)”系统，帮助您在写作过程中捕捉灵感、管理设定、追踪伏笔，并解决创作冲突。

## 1. 作品管理 (Works)

在左侧边栏，您可以管理您的所有创作项目。**在电脑端，侧边栏默认处于折叠状态以节省空间，您可以点击顶部的按钮展开。**

*   **创建作品**：在底部的输入框中输入新作品的名称，按回车键即可创建。
*   **切换作品**：点击列表中的作品名称，即可切换到该作品的编辑界面。
*   **修改作品图标**：
    *   在侧边栏**展开**状态下，点击作品名称左侧的图标，可以弹出图标选择器。
    *   支持选择内置颜色的书籍图标，也支持输入任意 Emoji 作为图标。
    *   *注：当侧边栏**收起**时，点击图标将直接切换到该作品，而不是修改图标。*
*   **重命名作品**：
    *   将鼠标悬停在作品名称上，点击出现的**铅笔图标**。
    *   或者直接**双击**作品名称。
    *   输入新名称后，按回车键或点击外部即可保存。
*   **删除作品**：
    *   将鼠标悬停在作品名称上，点击出现的**垃圾桶图标**。
    *   系统会显示红色的 **DELETE?** 提示，点击 **✓ (确认)** 将彻底删除该作品及其所有内容（章节、场景、角色等），点击 **✕ (取消)** 则放弃操作。
*   **排序作品**：按住作品名称左侧的**六点抓取图标**，上下拖动即可调整作品顺序。
*   **导入/导出**：
    *   **Export (导出)**：点击底部的 Export 按钮，将所有作品数据导出为 `.json` 文件备份。
    *   **Import (导入)**：点击底部的 Import 按钮，选择之前导出的 `.json` 文件，恢复数据。

## 2. 写作界面 (Writing Tab)

这是您进行创作的主要区域，包含章节列表和编辑器。

### 2.1 界面概览
*   **顶部导航栏 (Top Nav)**：
    *   **标签页切换**：根据不同的应用模式 (App Mode)，显示不同的功能标签页。
    *   **视图控制 (右上角)**：
        *   **Inspector (侧边栏开关)**：开启或关闭右侧的详情面板。
        *   **Focus Mode (专注模式)**：全屏沉浸式写作。
*   **右下角悬浮按钮**：
    *   **View Settings (视图设置)**：点击齿轮图标，可以调整字间距、页边距，切换 **App Mode (应用模式)**，开启 **Scroll Mode (滚动模式)** 或 **Disguise Mode (伪装模式)**，以及访问快捷键等功能。
    *   **回到顶部**：当编辑器内容较长时，点击可快速回到文档顶部。
*   **左侧边栏 (Sidebar)**：管理作品目录（卷、章、场景）。
*   **右侧面板 (Inspector)**：显示当前场景的客观事件参考、大纲、**Snapshots (历史快照)** 以及 **Inbox (收件箱)** 详情。

### 2.2 应用模式 (App Mode)
在编辑器右下角的 **View Settings** 菜单中，您可以切换应用模式，不同模式下顶部导航栏会显示不同的默认标签页：
*   **Design (设计)**：默认模式，专注于内容创作与世界构建。顶部导航栏显示：Writing (写作)、Timeline (时间轴)、Metro (故事地铁线)、World (世界)、Montage (蒙太奇看板)。
*   **Review (审阅)**：专注于阅读、修改与任务管理。顶部导航栏显示：Writing (写作)、Notes (笔记)、Block Descriptions (块目录)、Lenses (透镜)、Deadline (任务)。
*   **Manage (管理)**：专注于数据、导出与发布。顶部导航栏显示：Writing (写作)、Compile (编译)、Data Manager (数据管理)、Publish (发布管理)。

### 2.3 章节与场景管理
*   **创建章节**：点击左侧章节列表底部的 "+ New Chapter" 按钮。
    *   *注：新建章节时，系统会自动为其创建一个默认场景 (New Scene) 和一个初始文本块，方便您直接开始写作。*
*   **创建场景**：在某个章节下，点击 "+ New Scene" 按钮。
    *   *注：新建场景时，系统会自动创建一个初始文本块。*
*   **章节概览视图**：在左侧边栏点击某个章节名称（而不是场景），右侧编辑器将显示该章节的**场景列表 (Scenes List)** 和**角色出场矩阵 (Character Appearance Matrix)**，方便您以表格形式管理角色在各场景的出场状态及备注。
*   **重命名**：双击章节或场景名称进行重命名。
*   **排序**：拖动章节或场景标题，可以调整它们的顺序。场景也可以在不同章节之间拖动。
*   **删除**：点击章节或场景右侧的**垃圾桶图标**，并点击 **✓ (确认)**。
*   **归档章节 (Archive)**：点击章节右侧的**归档图标**。归档后的章节将被隐藏且变为只读。

### 2.4 文本编辑
*   **普通文本**：在编辑器中直接输入文本。每个段落都是一个独立的块 (Block)。
*   **极致平滑的滚动体验**：编辑器底层采用了同步物理测量 (`getBoundingClientRect` + `useLayoutEffect`) 技术。无论您在多长的文本中点击、切换段落或进行编辑，系统都能精确到亚像素级进行滚动补偿。彻底消除了画面闪烁和跳动，让文字“稳如泰山”，提供极致流畅的写作体验。同时底部保留了充足的缓冲空白 (Scroll Past End)，确保最后一段也能舒适编辑。
*   **复制场景文本 (Copy Scene Text)**：位于编辑器顶部的操作栏（Find 按钮和 Inspector 按钮之间），一键复制当前场景的所有正文内容。
*   **查找与替换 (Find & Replace)**：点击编辑器顶部的 Find 按钮，开启查找与替换工具栏。
*   **块操作菜单**：将鼠标悬停在文本块上，块的**左下方**会出现操作按钮：
    *   **Add Text**：插入普通文本块。
    *   **Add Lens**：插入透镜块。
    *   **Compare & Edit (对比与编辑)**：点击此按钮（GitCompare 图标），会弹出一个双栏对比编辑窗口。左侧为原始版本，右侧为草稿/AI润色版本。您可以在两边同时编辑，底部会实时显示差异高亮 (Diff)。编辑完成后，您可以选择接受原文、接受草稿，或者保存并保持对比状态（此时块的右上角会显示蓝色的 "Comparing" 徽章）。
    *   **Delete Block**：删除当前块。
*   **向上合并 (Merge Up)**：点击块顶部的 `↑ Merge Up` 按钮，可将当前块与上一个文本块合并。
*   **块描述与完成状态**：将鼠标悬停在普通文本块上，右侧会出现按钮：
    *   **块描述 (气泡图标)**：添加批注或大纲描述。
    *   **完成状态 (打钩图标)**：切换该文本块的完成状态。

### 2.5 Lens (透镜) 系统
Lens 用于在正文中嵌入元数据、设定或思考。
*   **颜色分类**：🔴 冲突/问题、🔵 设定/世界观、🟢 伏笔/线索、🟡 灵感/备注、🟣 角色发展、⚫ 黑色 (隐藏内容)。
*   **黑色 Lens**：内容默认透明，只有点击输入框时才会显示，用于隐藏剧透。
*   **Lens 操作**：支持切换颜色、固定 (Pin)、移除 (保留文本) 以及**自动双向链接**。

### 2.6 客观参考 (Objective Reference)
在 Inspector 面板中，系统会自动聚合当前场景所关联的所有**客观事件 (Timeline Events)**。
*   **人物行为**：显示该场景涉及的人物在对应时间点客观做了什么。
*   **一致性保障**：无论从哪个视角（场景）描写，底层的客观事实保持绝对一致，防止“吃书”。

### 2.7 专注、滚动与伪装模式
*   **专注模式 (Focus Mode)**：隐藏所有导航和侧边栏，文本区域收窄并居中，提供沉浸式体验。按钮位于顶部导航栏右侧。
*   **滚动模式 (Scroll Mode)**：在编辑器右下角的 **View Settings** 中开启。开启后，编辑器将连续显示当前章节下的所有场景，您可以无缝滚动阅读和编辑整个章节的内容。每个场景上方会显示其标题，并提供一键复制该场景文本的按钮。
*   **伪装模式 (Disguise Mode)**：隐藏所有操作按钮和侧边栏，将文本排版为纯净的连续文章，看起来不像在写作。开关位于右下角 **View Settings (视图设置)** 菜单中。按 `Esc` 键可快速退出。
    *   *注：当前伪装模式处于初步实现阶段，界面布局及交互细节（如编辑时的页面宽度适配）需要进一步优化调整。*

### 2.8 历史快照 (Snapshots)
在右侧面板 (Inspector) 中，您可以切换到 **Snapshots (历史快照)** 标签页。
*   **创建快照**：点击 "Create Snapshot" 按钮，为当前场景保存一个版本备份。
*   **版本管理**：列表显示所有历史快照，支持重命名和删除。
*   **差异对比与恢复**：点击任意历史快照，下方会显示该快照与当前场景内容的具体差异 (Diff)。您可以一键复制快照内容，或点击 "Restore This Version" 将场景恢复到该历史版本。

## 3. 块目录 (Block Descriptions Tab)

按照“章节 > 场景”的结构，汇总显示所有带有**块描述**的文本块，支持快速跳转。

## 4. 透镜概览 (Lenses Tab)

以瀑布流形式展示全书所有的 Lens，支持按章节和颜色筛选，并提供详情编辑面板（包含私有笔记）。

## 5. 蒙太奇看板 (Montage Board Tab)

*   **叙事流管理**：以垂直时间轴形式展示全书的章节与场景。
*   **事件关联**：每个场景卡片内置搜索多选框，可快速关联客观事件。
*   **角色行为矩阵 (Matrix View)**：关联事件后，场景下方会自动展开矩阵视图（行：角色，列：事件），方便您在宏观层面批量填入和修改每个角色在不同事件中的动作。
*   **非线性剪辑**：通过调整事件与场景的关联，重新构思故事结构。

## 6. 时间轴与事件 (Timeline Tab)

LensWriter 采用“客观事件驱动”模型，将“客观发生的事实”与“主观叙事的重组”彻底剥离。

*   **客观事件 (TimelineEvent)**：记录在某个时间、地点，特定人物做的客观动作。
*   **表格视图 (Table View)**：
    *   **自定义列管理**：点击表头的设置图标（Settings2）可打开列管理面板。支持**显隐控制**（勾选/取消勾选）、**排序调整**（拖拽排序）以及**一键重置**。
    *   **布局优化**：支持**超长滚动 (Overscrolling)**，底部预留空白空间以便将数据居中管理；表格在大屏幕下自动**水平居中**。
    *   **折叠搜索框**：顶部搜索框默认折叠，点击图标展开，有内容时保持展开。
    *   **统一搜索栏**：顶部提供全行搜索框，支持对事件标题和描述进行模糊查询。
    *   **折叠筛选面板**：点击搜索框末尾的 **Filters** 按钮可展开详细筛选。
    *   **多维筛选**：支持按 **角色 (Characters)**、**地点 (Locations)**、**标签 (Tags)** 进行多选过滤。
    *   **范围查询**：支持对 **开始时间 (Start Time)** 和 **时长 (Duration)** 进行最小值和最大值的范围筛选。
    *   **操作**：点击 **Apply Filters** 生效，点击 **Reset All** 清空所有筛选条件。
*   **编年史视图 (Chronology)**：按时间顺序排列所有事件，支持多维筛选。
*   **事件池 (Event Pool)**：管理所有未挂载到场景的备选事件。
*   **标签系统**：为事件添加自定义标签（如：主线、支线、伏笔）。

## 7. 世界观、角色与地点 (World Tab)

在顶部的 **World** 标签页中，您可以管理作品的底层设定。

*   **角色管理 (Characters)**：
    *   添加、编辑、排序和删除角色。
    *   **自定义字段**：配置角色的属性（如年龄、阵营等）。
    *   **登场追踪 (Appearance)**：查看角色在哪些场景出现过。
*   **地点管理 (Locations)**：
    *   记录故事发生的地理位置及其描述。
    *   关联到客观事件，构建空间一致性。
*   **世界观设定 (World)**：管理更宏观的设定条目。

## 8. 任务与日历 (Deadline Tab)

LensWriter 提供了两种维度的任务管理视图：

### 8.1 全局视图 (Global Deadline)
*   **进入方式**：点击左侧边栏顶部的 **日历图标 (Deadline)**。
*   **功能**：显示所有作品的任务进度。在列表中点击某个作品，会自动切换到该作品的**局部视图**。

### 8.2 局部视图 (Local Deadline)
*   **进入方式**：点击顶部导航栏的 **Deadline** 标签。
*   **功能**：专注于当前正在编辑的作品。界面隐藏了作品标题，直接显示 **"Total Count" (总字数)**。

### 8.3 核心功能
*   **设定目标**：为章节输入目标字数 (Goal) 即可创建任务。
*   **日历规划**：将任务卡片拖拽到日历日期上分配截止日期 (DDL)。
*   **进度追踪**：卡片实时显示当前字数与目标字数的完成百分比。

### 8.4 每日字数统计 (Word Count Statistics)
*   **日历统计**：日历中的每个日期方格会显示当天的**累计总字数**和**净变化量 (Net)**。
    *   **累计总字数**：当天正文区域的总字数。
    *   **净变化量**：当天字数的增减情况（包含删除的字数）。
*   **趋势折线图**：点击日历右上角的**图表图标**，可以展开/折叠字数趋势折线图，直观查看字数变化。
*   **重置与编辑**：
    *   **重置 (Reset)**：点击日历右上角的 **Undo (重置)** 按钮，可以清空今日的统计数据。
    *   **任意日期管理**：点击日历上的任意日期，会弹出编辑窗口：
        *   **Reset**：重置该日期的统计数据为 0。
        *   **Save (手动编辑)**：手动输入该日的总字数，保存后该日期旁会显示黄色星号 (`*`)，表示该数据为手动修改。
    *   **安全确认**：重置操作采用内联二次确认机制，防止误触。

## 9. 编译与导出 (Compile Tab)

将作品导出为结构化的 Word 文档 (.docx)。
*   **选择内容**：支持选中整部作品、当前章节、当前场景或手动勾选。
*   **预览与复制**：实时预览纯文本内容，支持一键复制。黑色隐藏 Lens 的内容会被自动排除。
*   **导出文档**：生成带有一级标题结构的 .docx 文件。

## 10. 收件箱与快速记录 (Inbox Tab)

用于捕捉碎片化的灵感和素材。

*   **快速记录 (Quick Capture)**：点击右下角的闪电图标，快速输入灵感并保存到收件箱。
*   **收件箱标签页 (Inbox Tab)**：管理所有未分类的灵感，支持编辑、删除和转换为正式设定。

## 11. 移动端适配 (Mobile Adaptation)

*   **导航优化**：主要标签页切换移动到屏幕底部。
*   **全屏体验**：章节目录、编辑界面、角色详情和 Lens 详情均采用全屏设计。
*   **交互优化**：防止缩放、锁定横向滚动，确保单手操作流畅。
*   **界面修复**：针对手机浏览器工具栏遮挡问题，采用了动态视口高度 (100dvh) 技术，并为底部按钮增加了安全区避让 (Safe Area Inset)，确保在所有手机型号上均能正常操作。

## 12. 自动保存与数据同步

### 12.1 自动保存 (Auto-Save)
所有修改都会实时保存到浏览器的 **本地存储 (LocalStorage)** 中。建议定期使用 **Export** 功能或开启 **云端同步** 进行备份。

### 12.2 数据同步与备份 (Sync & Backup)
LensWriter 提供多种方式保障您的数据安全：

*   **本地存储 (LocalStorage)**：默认保存，实时同步。
*   **本地文件夹备份 (Local Folder Backup)**：仅支持桌面端 Chrome/Edge，实时同步到指定文件夹。
*   **Supabase 云端同步 (Cloud Sync)**：通过 Supabase 实现多端实时同步。
*   **手动导入导出 (Import/Export)**：生成 `.json` 文件备份或恢复数据。

## 13. 快捷键 (Keyboard Shortcuts)

为了提高写作效率，LensWriter 支持以下快捷键操作（Windows 使用 `Ctrl`，macOS 使用 `Cmd`）：

| 快捷键 | 功能 |
| :--- | :--- |
| `Ctrl/Cmd` + `F` | 打开/关闭查找与替换工具栏 |
| `Ctrl/Cmd` + `Shift` + `M` | 切换应用模式 (App Mode) |
| `Ctrl/Cmd` + `/` | 切换当前块的透镜 (Lens) 状态 |
| `Ctrl/Cmd` + `Enter` | 在当前块下方添加新文本块 |
| `Ctrl/Cmd` + `Z` | 撤销 (Undo) |
| `Ctrl/Cmd` + `Shift` + `Z` 或 `Y` | 重做 (Redo) |
| `Tab` | 移动焦点至下一个块或场景 |
| `Shift` + `Tab` | 移动焦点至上一个块或场景 |
| `/` (块首) | 打开斜杠命令菜单 (Slash Command Menu) |

## 14. 发布管理 (Publish Management)

LensWriter 提供了强大的发布管理功能，帮助您在多个平台（如起点、知乎等）同步作品进度，并追踪每个章节的版本。

### 14.1 核心概念
*   **章节快照 (Chapter Snapshot)**：记录某一时刻该章节的完整内容。您可以为同一章节创建多个快照（如：初稿、精修、最终版）。
*   **平台追踪 (Platform Tracking)**：为每个发布平台（如“起点中文网”）建立独立的追踪记录。

### 14.2 界面布局
发布管理界面采用全屏双栏设计：
*   **左侧：书籍目录与快照**
    *   显示全书章节列表。
    *   每个章节旁显示当前已有的**快照数量**。
    *   点击章节可展开查看其所有历史快照，并支持删除旧快照。
    *   点击顶部的 `+` 按钮可为当前选中的章节创建新的版本快照。
*   **右侧：发布平台与状态**
    *   列出所有已添加的发布平台。
    *   每个平台卡片可**折叠/展开**，方便管理多个平台。
    *   **状态标识**：
        *   **Published (已发布)**：显示绿色标签，记录最后一次发布的版本名称。
        *   **Needs Update (待更新)**：显示醒目的**橙色呼吸灯标签**，表示当前章节内容已修改，且新内容与已发布版本存在差异。
        *   **Not Published (未发布)**：表示该章节尚未在该平台发布。

### 14.3 发布工作流
1.  **创建快照**：在左侧选中章节，点击 `+` 创建一个新版本。
2.  **添加平台**：点击右侧顶部的 `+ Add Platform`，输入平台名称。
3.  **发布/更新**：
    *   点击章节旁的 `Publish` 或 `Update` 按钮，将最新的快照标记为已发布。
    *   如果显示 `UPDATE AVAILABLE`，点击旁边的**刷新图标**可以查看**差异对比 (Diff View)**，直观看到当前内容与已发布版本的具体修改点（精确到场景和文本块）。
    *   在差异视图中确认无误后，点击 `Confirm Update` 完成状态同步。

## 15. 剧本写作 (Script Tab)

LensWriter 提供了一个专门的剧本写作模式，让您可以像文字冒险游戏一样快速编写对话和旁白。

### 15.1 写作区 (Writing Area)
*   **角色选择**：使用快捷键 `Alt + 1`, `Alt + 2` 等快速选择当前作品中的角色。`Alt + 0` 切换为旁白模式。按 `Esc` 键可取消选择。
*   **快速输入**：在底部输入框输入内容，按 `Enter` 快速发送，按 `Shift + Enter` 换行。
*   **修改内容**：鼠标悬停在已发送的对话或旁白上，点击出现的**铅笔图标**，即可将内容重新载入输入框进行修改。修改完成后按 `Enter` 保存。
*   **删除内容**：鼠标悬停在已发送的行上，点击**垃圾桶图标**即可删除。
*   **打字机回放**：点击顶部的 `Play` 按钮，可以像视觉小说游戏一样，逐字回放当前编写的剧本内容。
*   **保存草稿**：点击 `Save Draft`，可以输入一个标题（默认为“未命名剧本”）将当前内容保存到管理区。

### 15.2 管理区 (Management Area)
*   **草稿列表**：查看所有已保存的剧本草稿。
*   **修改标题**：鼠标悬停在草稿标题上，点击**铅笔图标**即可直接修改标题。
*   **载入草稿**：点击草稿右上角的**载入图标**，可以将该草稿的全部内容重新载入到写作区，继续进行编写或修改。
*   **插入到场景**：选择目标场景后，点击 `Insert to Scene`，剧本内容会自动转换为普通文本并插入到指定场景的末尾（对话会自动带上角色名）。
*   **删除草稿**：点击**垃圾桶图标**删除不需要的草稿。
