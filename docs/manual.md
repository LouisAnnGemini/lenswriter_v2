# LensWriter 功能说明书

LensWriter 是一款专为作家和创作者设计的非线性写作工具，它通过独特的“透镜 (Lens)”系统，帮助您在写作过程中捕捉灵感、管理设定、追踪伏笔，并解决创作冲突。

## 1. 作品管理 (Works)

在左侧边栏，您可以管理您的所有创作项目。**在电脑端，侧边栏默认处于折叠状态以节省空间，您可以点击顶部的按钮展开。**

*   **创建作品**：在底部的输入框中输入新作品的名称，按回车键即可创建。
*   **切换作品**：点击列表中的作品名称，即可切换到该作品的编辑界面。
*   **修改作品图标**：
    *   在侧边栏**展开**状态下，点击作品名称左侧的图标，可以弹出图标选择器。
    *   支持选择内置颜色的书籍图标，也支持输入任意 Emoji 作为图标。
    *   *注：当侧边栏**收起**时，点击图标将直接切换到该作品，而不是修改图标。*
*   **重命名作品**：
    *   将鼠标悬停在作品名称上，点击出现的**铅笔图标**。
    *   或者直接**双击**作品名称。
    *   输入新名称后，按回车键或点击外部即可保存。
*   **删除作品**：
    *   将鼠标悬停在作品名称上，点击出现的**垃圾桶图标**。
    *   系统会显示红色的 **DELETE?** 提示，点击 **✓ (确认)** 将彻底删除该作品及其所有内容（章节、场景、角色等），点击 **✕ (取消)** 则放弃操作。
*   **排序作品**：按住作品名称左侧的**六点抓取图标**，上下拖动即可调整作品顺序。
*   **导入/导出**：
    *   **Export (导出)**：点击底部的 Export 按钮，将所有作品数据导出为 `.json` 文件备份。
    *   **Import (导入)**：点击底部的 Import 按钮，选择之前导出的 `.json` 文件，恢复数据。

## 2. 写作界面 (Writing Tab)

这是您进行创作的主要区域，包含章节列表和编辑器。

### 2.1 界面概览
*   **顶部导航栏 (Top Nav)**：
    *   **标签页切换**：Writing (写作), Board (看板), World (世界), Deadline (任务), Compile (编译), Inbox (收件箱)。
    *   **视图控制 (右上角)**：
        *   **Inspector (侧边栏开关)**：开启或关闭右侧的详情面板。
        *   **Focus Mode (专注模式)**：全屏沉浸式写作。
*   **右下角悬浮按钮**：
    *   **View Settings (视图设置)**：点击齿轮图标，可以调整字间距、页边距，以及切换 **App Mode (应用模式)** 或开启 **Disguise Mode (伪装模式)**。
    *   **回到顶部**：当编辑器内容较长时，点击可快速回到文档顶部。
*   **左侧边栏 (Sidebar)**：管理作品目录（卷、章、场景），顶部包含 **Inbox (收件箱)** 入口。
*   **右侧面板 (Inspector)**：显示当前场景的客观事件参考、大纲以及 **Inbox (收件箱)** 详情。

### 2.2 应用模式 (App Mode)
在编辑器右下角的 **View Settings** 菜单中，您可以切换应用模式：
*   **Design Mode (设计模式)**：默认模式，侧边栏显示完整的章节和场景目录，顶部导航栏显示设计相关的标签页（如 Lenses, Timeline, World）。
*   **Management Mode (管理模式)**：侧边栏仅显示章节，顶部导航栏显示管理相关的标签页（如 Inbox, Deadline, Compile）。

### 2.3 章节与场景管理
*   **创建章节**：点击左侧章节列表底部的 "+ New Chapter" 按钮。
    *   *注：新建章节时，系统会自动为其创建一个默认场景 (New Scene) 和一个初始文本块，方便您直接开始写作。*
*   **创建场景**：在某个章节下，点击 "+ New Scene" 按钮。
    *   *注：新建场景时，系统会自动创建一个初始文本块。*
*   **章节概览视图**：在左侧边栏点击某个章节名称（而不是场景），右侧编辑器将显示该章节的**场景列表 (Scenes List)** 和**角色出场总结 (Character Appearances)**，方便您从宏观角度审视整个章节的结构。
*   **重命名**：双击章节或场景名称进行重命名。
*   **排序**：拖动章节或场景标题，可以调整它们的顺序。场景也可以在不同章节之间拖动。
*   **删除**：点击章节或场景右侧的**垃圾桶图标**，并点击 **✓ (确认)**。
*   **归档章节 (Archive)**：点击章节右侧的**归档图标**。归档后的章节将被隐藏且变为只读。

### 2.3 文本编辑
*   **普通文本**：在编辑器中直接输入文本。每个段落都是一个独立的块 (Block)。
*   **复制场景文本 (Copy Scene Text)**：位于编辑器顶部的操作栏（Find 按钮和 Inspector 按钮之间），一键复制当前场景的所有正文内容。
*   **查找与替换 (Find & Replace)**：点击编辑器顶部的 Find 按钮，开启查找与替换工具栏。
*   **块操作菜单**：将鼠标悬停在文本块上，块的**左下方**会出现操作按钮：
    *   **Add Text**：插入普通文本块。
    *   **Add Lens**：插入透镜块。
    *   **Delete Block**：插入普通文本块。
*   **向上合并 (Merge Up)**：点击块顶部的 `↑ Merge Up` 按钮，可将当前块与上一个文本块合并。
*   **块描述与完成状态**：将鼠标悬停在普通文本块上，右侧会出现按钮：
    *   **块描述 (气泡图标)**：添加批注或大纲描述。
    *   **完成状态 (打钩图标)**：切换该文本块的完成状态。

### 2.4 Lens (透镜) 系统
Lens 用于在正文中嵌入元数据、设定或思考。
*   **颜色分类**：🔴 冲突/问题、🔵 设定/世界观、🟢 伏笔/线索、🟡 灵感/备注、🟣 角色发展、⚫ 黑色 (隐藏内容)。
*   **黑色 Lens**：内容默认透明，只有点击输入框时才会显示，用于隐藏剧透。
*   **Lens 操作**：支持切换颜色、固定 (Pin)、移除 (保留文本) 以及**自动双向链接**。

### 2.5 客观参考 (Objective Reference)
在 Inspector 面板中，系统会自动聚合当前场景所关联的所有**客观事件 (Timeline Events)**。
*   **人物行为**：显示该场景涉及的人物在对应时间点客观做了什么。
*   **一致性保障**：无论从哪个视角（场景）描写，底层的客观事实保持绝对一致，防止“吃书”。

### 2.6 专注与伪装模式
*   **专注模式 (Focus Mode)**：隐藏所有导航和侧边栏，文本区域收窄并居中，提供沉浸式体验。按钮位于顶部导航栏右侧。
*   **伪装模式 (Disguise Mode)**：隐藏所有操作按钮和侧边栏，将文本排版为纯净的连续文章，看起来不像在写作。开关位于右下角 **View Settings (视图设置)** 菜单中。按 `Esc` 键可快速退出。

## 3. 收件箱与快速记录 (Inbox & Quick Capture)

用于捕捉碎片化的灵感和素材。

*   **快速记录 (Quick Capture)**：点击右下角的闪电图标，快速输入灵感并保存到收件箱。
*   **收件箱标签页 (Inbox Tab)**：管理所有未分类的灵感，支持编辑、删除和转换为正式设定。
*   **侧边面板 (Inbox Panel)**：在写作时，通过右侧 Inspector 面板快速查看和引用收件箱内容。

## 4. 看板与管理 (Board Tab)

在顶部的 **Board** 标签页中，您可以进行多维度的作品梳理。点击 **Board** 按钮旁边的下拉箭头，可以切换不同的看板视图：

*   **块目录 (Block Descriptions)**：按照“章节 > 场景”的结构，汇总显示所有带有**块描述**的文本块，支持快速跳转。
*   **透镜概览 (Lenses)**：以瀑布流形式展示全书所有的 Lens，支持按章节和颜色筛选，并提供详情编辑面板（包含私有笔记）。
*   **蒙太奇看板 (Montage Board)**：
    *   **叙事流管理**：以垂直时间轴形式展示全书的章节与场景。
    *   **事件关联**：每个场景卡片内置搜索多选框，可快速关联客观事件。
    *   **角色行为矩阵 (Matrix View)**：关联事件后，场景下方会自动展开矩阵视图（行：角色，列：事件），方便您在宏观层面批量填入和修改每个角色在不同事件中的动作。
    *   **非线性剪辑**：通过调整事件与场景的关联，重新构思故事结构。

## 5. 时间轴与事件 (Timeline & Events)

LensWriter 采用“客观事件驱动”模型，将“客观发生的事实”与“主观叙事的重组”彻底剥离。

*   **客观事件 (TimelineEvent)**：记录在某个时间、地点，特定人物做的客观动作。
*   **表格视图 (Table View)**：
    *   **自定义列管理**：点击表头的设置图标（Settings2）可打开列管理面板。支持**显隐控制**（勾选/取消勾选）、**排序调整**（拖拽排序）以及**一键重置**。
    *   **布局优化**：支持**超长滚动 (Overscrolling)**，底部预留空白空间以便将数据居中管理；表格在大屏幕下自动**水平居中**。
    *   **折叠搜索框**：顶部搜索框默认折叠，点击图标展开，有内容时保持展开。
    *   **统一搜索栏**：顶部提供全行搜索框，支持对事件标题和描述进行模糊查询。
    *   **折叠筛选面板**：点击搜索框末尾的 **Filters** 按钮可展开详细筛选。
    *   **多维筛选**：支持按 **角色 (Characters)**、**地点 (Locations)**、**标签 (Tags)** 进行多选过滤。
    *   **范围查询**：支持对 **开始时间 (Start Time)** 和 **时长 (Duration)** 进行最小值和最大值的范围筛选。
    *   **操作**：点击 **Apply Filters** 生效，点击 **Reset All** 清空所有筛选条件。
*   **编年史视图 (Chronology)**：按时间顺序排列所有事件，支持多维筛选。
*   **事件池 (Event Pool)**：管理所有未挂载到场景的备选事件。
*   **标签系统**：为事件添加自定义标签（如：主线、支线、伏笔）。

## 6. 世界观、角色与地点 (World Tab)

在顶部的 **World** 标签页中，您可以管理作品的底层设定。

*   **角色管理 (Characters)**：
    *   添加、编辑、排序和删除角色。
    *   **自定义字段**：配置角色的属性（如年龄、阵营等）。
    *   **登场追踪 (Appearance)**：查看角色在哪些场景出现过。
*   **地点管理 (Locations)**：
    *   记录故事发生的地理位置及其描述。
    *   关联到客观事件，构建空间一致性。
*   **世界观设定 (World)**：管理更宏观的设定条目。

## 7. 任务与日历 (Deadline Tab)

LensWriter 提供了两种维度的任务管理视图：

### 7.1 全局视图 (Global Deadline)
*   **进入方式**：点击左侧边栏顶部的 **日历图标 (Deadline)**。
*   **功能**：显示所有作品的任务进度。在列表中点击某个作品，会自动切换到该作品的**局部视图**。

### 7.2 局部视图 (Local Deadline)
*   **进入方式**：点击顶部导航栏的 **Deadline** 标签。
*   **功能**：专注于当前正在编辑的作品。界面隐藏了作品标题，直接显示 **"Total Count" (总字数)**。

### 7.3 核心功能
*   **设定目标**：为章节输入目标字数 (Goal) 即可创建任务。
*   **日历规划**：将任务卡片拖拽到日历日期上分配截止日期 (DDL)。
*   **进度追踪**：卡片实时显示当前字数与目标字数的完成百分比。

## 8. 编译与导出 (Compile Tab)

将作品导出为结构化的 Word 文档 (.docx)。
*   **选择内容**：支持选中整部作品、当前章节、当前场景或手动勾选。
*   **预览与复制**：实时预览纯文本内容，支持一键复制。黑色隐藏 Lens 的内容会被自动排除。
*   **导出文档**：生成带有一级标题结构的 .docx 文件。

## 9. 移动端适配 (Mobile Adaptation)

*   **导航优化**：主要标签页切换移动到屏幕底部。
*   **全屏体验**：章节目录、编辑界面、角色详情和 Lens 详情均采用全屏设计。
*   **交互优化**：防止缩放、锁定横向滚动，确保单手操作流畅。
*   **界面修复**：针对手机浏览器工具栏遮挡问题，采用了动态视口高度 (100dvh) 技术，并为底部按钮增加了安全区避让 (Safe Area Inset)，确保在所有手机型号上均能正常操作。

## 10. 自动保存 (Auto-Save)

所有修改都会实时保存到浏览器的 **本地存储 (LocalStorage)** 中。建议定期使用 **Export** 功能或开启 **云端同步** 进行备份。

## 11. 数据同步与备份 (Sync & Backup)

LensWriter 提供多种方式保障您的数据安全：

### 11.1 本地存储 (LocalStorage)
*   **默认保存**：所有修改实时保存到浏览器本地，无需手动点击。
*   **局限性**：清除浏览器缓存或更换设备会导致数据丢失。

### 11.2 本地文件夹备份 (Local Folder Backup)
*   **功能**：将数据实时同步到您电脑上的一个特定文件夹中。
*   **要求**：仅支持 **桌面端 (Desktop)** 的 Chrome 或 Edge 浏览器。
*   **操作**：点击左下角设置图标 -> 选择本地文件夹 -> 授权访问。

### 11.3 Supabase 云端同步 (Cloud Sync)
*   **功能**：通过 Supabase 数据库实现多端实时同步（手机、平板、电脑）。
*   **要求**：需要您拥有一个 Supabase 项目，并创建名为 `app_state` 的表。
*   **表结构**：
    *   `id`: 类型为 `uuid` (主键)。
    *   `state`: 类型为 `jsonb`。
*   **操作**：在设置菜单中输入您的 Supabase URL 和 Anon Key，开启同步开关。
*   **状态监控**：设置菜单中会实时显示同步状态（Syncing / Success / Error）。

### 11.4 手动导入导出 (Import/Export)
*   **导出**：生成 `.json` 文件，建议定期手动备份。
*   **导入**：从备份文件恢复数据。
